# GEMINI.md

## Directory Overview
This directory contains instructional materials for the course "Calculus 2 for Engineers" (สอนโดย รศ.ดร.พงศกร สุนทรายุทธ์). The repository acts as a centralized location for accessing lecture notes and practice exercises. The main topics covered include polar coordinates, vector-valued functions, lines and planes in 3D space, calculus of several variables, and multiple integrals.

## Key Files
*   `README.md`: Provides the course description in both Thai and English. It also serves as an index, containing download links that map each topic/chapter to its corresponding PDF file.
*   `materials/ch_01.pdf` - `materials/ch_07.pdf`: Lecture notes and slides for chapters 1 through 7.
*   `materials/ex_mid.pdf`: Midterm exam exercises.
*   `materials/ex_final.pdf`: Final exam exercises.

## Usage
The contents of this directory are intended for students to download and review course materials. Users should consult the `README.md` file to identify the appropriate chapter for their current study topic and open the corresponding PDF file located in the `materials/` directory.
